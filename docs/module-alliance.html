alliance

A container for the alliance ESI endpoints. You should not require this module directly, as it technically returns a factory function that requires an internal API. Instead an instance is automatically exposed when the module:eve_swagger_interface is loaded and configured.

(static) get(id) → {external:Promise}

Source:
  • alliance.js, line 88
See:
  • https://esi.tech.ccp.is/latest/#!/Alliance/get_alliances_alliance_id

Get the public info of the alliance from the ESI endpoint. This makes an HTTP GET request to alliances/{id}/. The request is returned as an asynchronous Promise that resolves to an object parsed from the response JSON model. An example value looks like:

{
  "alliance_name": "C C P Alliance",
  "date_founded": "2016-06-26T21:00:00Z",
  "executor_corp": 98356193,
  "ticker": "<C C P>"
}

This promisifies a call to the generated AllianceApi.getAlliancesAllianceId.
