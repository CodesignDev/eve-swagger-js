Module: universe

A container for the universe ESI endpoints. You should not require this module directly, as it technically returns a factory function that requires an internal API. Instead an instance is automatically exposed when the module:eve_swagger_interface is loaded and configured.

(static) getCategories() → {external:Promise}

Get all item categories in the universe from the ESI endpoint. This makes an HTTP GET request to /universe/categories/. The request is returned as an asynchronous Promise that resolves to an array parsed from the response JSON model. An example value looks like:

[
  1,
  2,
  3
]

(static) getNamesOf(ids) → {external:Promise}

Look up the name and type of the list of ids the ESI endpoint. The ids can correspond to characters, corporations, alliances, stations, solar systems, constellations, regions, and types. This makes an HTTP POST request to /universe/names/. The request is returned as an asynchronous Promise that resolves to an object parsed from the response JSON model. An example value looks like:

[
  {
    "category": "character",
    "id": 95465499,
    "name": "CCP Bartender"
  },
  {
    "category": "solar_system",
    "id": 30000142,
    "name": "Jita"
  }
]

This promisifies a call to the generated UniverseApi.postUniverseNames.
