Api

Api

Api creates a shared, internal ApiProvider and then lazily instantiates all specific modules as needed. The Api instance is also a function that can be invoked to create a new Api instance with a different configuration.

Constructor

new Api(service, source, agent, language, timeout, maxConcurrent, minTime)

Source:
  • Api.js, line 40
See:
  • https://esi.tech.ccp.is/latest

Create a new Api with the given configuration provided in a single object map. If no argument is provided, the defaults are used.

Parameters:
Name Type Description
service String

URL to the ESI service, defaults to 'https://esi.tech.ccp.is/latest'.
source String

Data source used, defaults to tranquility.
agent String

Custom user agent string to send with each request, which defaults to this project but really should be set for your app
language String

Language character code, defaults to en-us
timeout Number

Request timeout in milliseconds, defaults to 6000
maxConcurrent Number

Maximum number of requests running at once, defaults to 0 (unlimited)
minTime Number

Minimum time before launching another request (in milliseconds), defaults to 0 (no rate limiting)
