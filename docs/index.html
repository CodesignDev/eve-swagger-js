Eve Swagger Interface

An OpenAPI for EVE Online

JavaScript client for ESI compatible with NodeJS. The client returns a developer friendly set of functions that return promises resolving to the response from ESI endpoint. The underlying client library is generated by Swagger Codegen. Its documentation can be found here, which is primarily useful for details on the response data models. Versioning will follow the version reported by ESI as best as possible.

  • API version: 0.3.2
  • Package version: 0.3.2

Installation

For Node.js

Currently, this library has not been published to the npm repository. However, it can still be depended on in NodeJS projects by including a dependency in package.json using a GitHub URL:

"dependencies": {
   "eve_swagger_interface": "git://github.com/lhkbob/eve-swagger-js.git",
}

Getting Started

Please follow the installation instruction and execute the following JS code:

var ESI = require('eve_swagger_interface');

ESI.Alliance.getAll().then(function(alliances) {
    console.log(alliances);
}).catch(function(error) {
    console.error(error);
});
