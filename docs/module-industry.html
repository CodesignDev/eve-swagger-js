Module: industry

A container for the industry ESI endpoints. You should not require this module directly, as it technically returns a factory function that requires an internal API. Instead an instance is automatically exposed when the module:eve_swagger_interface is loaded and configured.

Methods

(static) getFacilities() → {external:Promise}

Get all industry facilities from the ESI endpoint. This makes an HTTP GET request to /industry/facilities. The request is returned as an asynchronous Promise that resolves to an array parsed from the response JSON model. An example value looks like:

[
  {
    "facility_id": 60012544,
    "owner_id": 1000126,
    "region_id": 10000001,
    "solar_system_id": 30000032,
    "tax": 0.1,
    "type_id": 2502
  }
]

This promisifies a call to the generated IndustryApi.getIndustryFacilities.

(static) getSystemCostIndices() → {external:Promise}

Get cost indices for each solar system from the ESI endpoint. This makes an HTTP GET request to /industry/systems. The request is returned as an asynchronous Promise that resolves to an array parsed from the response JSON model. An example value looks like:

[
  {
    "cost_indices": [
      {
        "activity": "invention",
        "cost_index": 0.00480411064973412
      }
    ],
    "solar_system_id": 30011392
  }
]

This promisifies a call to the generated IndustryApi.getIndustryFacilities.
