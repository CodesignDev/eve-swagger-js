Window

Window

An api adapter that provides functions for modifying the character's in-game windows via functions in the user interface ESI endpoints. You should not usually instantiate this directly as its constructor requires an internal api instance.

contract(contractId) → {Promise}

Source:
  • api/character/ui/Window.js, line 65

Open the in-game contracts window to the specified contract via the ESI endpoint. This makes an HTTP POST request to /ui/openwindow/contract/. The request is returned as an asynchronous Promise that resolves to an empty object on success. This promisifies a call to the generated UserInterfaceApi.postUiOpenwindowContract.

newMail(settings) → {Promise}

Source:
  • api/character/ui/Window.js, line 94

Open the in-game mail window to send a new mail via the ESI endpoint. This makes an HTTP POST request to /ui/openwindow/contract/. The request is returned as an asynchronous Promise that resolves to an empty object on success. The new mail dialog is initialized based on the mail configuration passed in, an example of which looks like:

{
  "body": "string",
  "recipients": [
    0
  ],
  "subject": "string",
  "to_corp_or_alliance_id": 0,
  "to_mailing_list_id": 0
}

This promisifies a call to the generated UserInterfaceApi.postUiOpenwindowNewmail.

Parameters:
Name Type Description
settings Object

The initial state of the new mail
