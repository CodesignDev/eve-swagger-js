Module: insurance

A container for the insurance ESI endpoints. You should not require this module directly, as it technically returns a factory function that requires an internal API. Instead an instance is automatically exposed when the module:eve_swagger_interface is loaded and configured.

Methods

(static) getPrices(language) → {external:Promise}

Get insurance prices for all ship types from the ESI endpoint. This makes an HTTP GET request to /insurance/prices/. The request is returned as an asynchronous Promise that resolves to an array parsed from the response JSON model. An example value looks like:

[
  {
    "levels": [
      {
        "cost": 10,
        "name": "Basic",
        "payout": 20
      }
    ],
    "type_id": 1
  }
]

This promisifies a call to the generated InsuranceApi.getInsurancePrices.

Parameters:
Name Type Description
language String

Optional localization code for the response, which overrides the default language.
