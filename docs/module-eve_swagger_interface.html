Module: eve_swagger_interface

Hand-written Promisified wrapper around generated ESI code. This module returns a factory function that takes an optional datasource string and base URL. If any parameter is not provided, the defaults are used.

The factory function returns an object with properties that expose instantiations of each sub-module for the ESI groups.

Parameters:
Name Type Description
datasource String

The datasource string, defaults to 'tranquility'
baseURL String

The base URL of the ESI Swagger endpoint, which defaults to 'https://esi.tech.ccp.is/latest'
