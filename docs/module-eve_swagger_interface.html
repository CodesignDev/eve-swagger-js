eve_swagger_interface

Hand-written Promisified wrapper around generated ESI code. This module returns a factory function that takes an optional datasource string and base URL. If any parameter is not provided, the defaults are used.

The factory function returns an object with properties that expose instantiations of each sub-module for the ESI groups.

Source:
  • index.js, line 1
Parameters:
Name Type Description
opts.accessToken String

The default access token to provide on all requests; methods that require authentication provide an optional access token argument so that it can be overridden per request. Defaults to null
opts.baseURL String

The base URL of the ESI endpoint, which defaults to 'https://esi.tech.ccp.is/latest'
opts.datasource String

The datasource for the ESI endpoint, defaults to 'tranquility'
opts.language String

The default language locale code, defaults to en-us, but can be set to one of languages. Requests that support localization provide a language parameter so it can be overridden per request.
