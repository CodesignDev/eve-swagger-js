Contact

Contact

An api adapter that provides functions for modifying and removing a particular contact of a character, specified by id via functions in the contacts ESI endpoints.

You should not usually instantiate this directly as its constructor requires an internal api instance.

remove() → {Promise}

Source:
  • api/character/Contacts.js, line 36

Remove the contact the character's contact list via the ESI end point. This makes an HTTP DELETE request to /characters/{characterId}/contacts. The request is returned as an asynchronous Promise that resolves to an empty object from the response JSON model. This promisifies a call to the generated ContactsApi.deleteCharactersCharacterIdContacts.

Returns:

A Promise that resolves to the response of the request

Promise

update(standing, label) → {Promise}

Source:
  • api/character/Contacts.js, line 67

Update the character's contact to have the given settings via the ESI end point. This makes an HTTP PUT request to /characters/{characterId}/contacts. The request is returned as an asynchronous Promise that resolves to an empty object on success.

The updated contacts are set to unwatched. This promisifies a call to the generated ContactsApi.putCharactersCharacterIdContacts.

Parameters:
Name Type Default Description
standing Number

The standing for all contacts

Optional; custom label id to add contacts to. If not provided, the contact has its previous label removed
