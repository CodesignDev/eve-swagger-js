fleets

A container for the fleets ESI endpoints. You should not require this module directly, as it technically returns a factory function that requires an internal API. Instead an instance is automatically exposed when the module:eve_swagger_interface is loaded and configured.

Add a new sequad to a fleet's wing with the ESI endpoint. This makes an HTTP POST request to /fleets/{fleetId}/wings/{wingId}/squads. The request is returned as an asynchronous Promise that resolves to an object parsed from the response JSON model. An example value looks like:

{
  "squad_id": 123
}

This promisifies a call to the generated FleetsApi.postFleetsFleetIdWingsWingIdSquads.
