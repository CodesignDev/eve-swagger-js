Contacts

Contacts

An api adapter over the end points handling a character's contacts via functions in the contacts ESI endpoints. You should not usually instantiate this directly as its constructor requires an internal api instance.

This is a function class so instances of Contacts are functions and can be invoked directly, besides accessing its members. Its default function action is equivalent to get or all if no id is provided.

Methods

add(contacts, standing, label) → {Promise}

Source:
  • api/character/Contacts.js, line 218

Bulk add contacts with the same settings to the given character's contact list via the ESI end point. This makes an HTTP POST request to /characters/{characterId}/contacts. The request is returned as an asynchronous Promise that resolves to an array from the response JSON model. An example return value looks like:

[
  123,
  456
]

The added contacts are unwatched. This promisifies a call to the generated ContactsApi.postCharactersCharacterIdContacts.

Optional; custom label id to add contacts to

all(page) → {Promise}

Source:
  • api/character/Contacts.js, line 150

Get the contacts for the given character via the ESI end point. The page parameter controls which page is selected. This makes an HTTP GET request to /characters/{id}/contacts. The request is returned as an asynchronous Promise that resolves to an array parsed from the response JSON model. An example value looks like:

[
  {
    "contact_id": 123,
    "contact_type": "character",
    "is_blocked": false,
    "is_watched": true,
    "standing": 10
  }
]

Technically, this end point is paginated by ESI. If invoked without a page argument, the end point will be queried multiple times to fetch the entirety of groups. This promisifies a call to the generated ContactsApi.getCharactersCharacterIdContacts.

Parameters:
Name Type Default Description
page Number 0

Optional; the page of groups to return, starting with page 1. If not provided, all groups are returned.
Returns:

A Promise that resolves to the response of the request
